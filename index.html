Programmi

Gli unici sorgenti che, per varie ragioni, posso rilasciare (sotto licenza BSD).

C++/STL

  • instrumentation una libreria per il tracciamento delle chiamate a funzione (2007).
  • heap una libreria per il tracciamento delle allocazioni di memoria (2006).
  • cache.h (7KB), una cache a profondità variabile basata su STL, e relativo test case (3KB) (2001).

C++/MFC

  • ScreenSaver (11KB), ossatura di un salva-schermo e relativa finestra di configurazione, per VC 5.0. Modificare l'estensione del file eseguibile in .scr e copiare il file in %SystemRoot%\system32. Il salva-schermo sarà disponibile, tra gli altri, nella maschera delle proprietà dello schermo (1998).
  • FPSrv (43KB), un server di stampa per stampante fotografica digitale Sienna FotoPrint. Maggiori dettagli qui (1998).

C++/Win32

  • NTService (12KB), esempio di servizio NT, per VC 6.0. Per le istruzioni d'uso, avviare il programma con il flag -h. (2001).
  • Fake Format, un programma che simula la formattazione di un disco, utilizzato per indurre alcune persone ad attivare una politica di back-up dei propri dati. Sono disponibili il file sorgente (5KB) e l'eseguibile (9KB) (2004).
