Arduino IDE su Ubuntu

L’IDE di Arduino si può installare su Ubuntu dall’Ubuntu Software Center, tuttavia la versione presente negli archivi ufficiali è piuttosto vecchia (1.0.5). Ho installato la versione più recente su Ubuntu 14.04 64 bit in questo modo:

  • ho scaricato l’ultima versione disponibile dell’IDE dal sito ufficiale (nel mio caso arduino-1.6.5-r5-linux64.tar.xz);
  • ho scompattato il file e spostato il contenuto nella cartella opt da una finestra del terminale con i seguenti comandi: 
    cd
cd Downloads
tar -xpvf arduino-1.6.5-r5-linux64.tar.xz
sudo mv arduino-1.6.5-r5 /opt
  • l’IDE di Arduino si può già avviare con il comando: 
    /opt/arduino-1.6.5-r5/arduino
  • per comodità ho creato uno shortcut aggiungendo un file di testo denominato arduino.dekstop sul Desktop il cui contenuto è: 
    [Desktop Entry]
Type=Application
Name=Arduino IDE
GenericName=Integrated Development Environment
Comment=An IDE for Arduino-compatible electronics prototyping platforms
Exec=/opt/arduino-1.6.5-r5/arduino
Icon=/opt/arduino-1.6.5-r5/lib/arduino.png
Terminal=false
Categories=Development;IDE;Electronics;
MimeType=text/x-arduino
Keywords=embedded electronics;electronics;avr;microcontroller;
    e rendendolo eseguibile con il comando: 
    chmod +x ~/Desktop/arduino.desktop
  • ho infine concesso all’utente corrente i permessi per l’accesso alle porte USB con il comando: 
    sudo usermod -a -G dialout «nome-utente»
    Per rendere effettiva questa impostazione conviene chiudere la sessione corrente e/o riavviare il sistema operativo.

L’IDE di Arduino necessita del runtime di Java per funzionare correttamente. Se non è già presente nel sistema, lo si può installare con il comando sudo apt-get install openjdk-7-jre.
