Bluetooth, C++ e Android

There is no direct access to Bluetooth APIs from the NDK. The list of available NDK APIs is inside docs/STABLE-APIS.html from the NDK archive. But that doesn't mean that you shouldn't use Bluetooth from the NDK. You can directly call Java Android APIs and manipulate Java objects from C/C++ using JNI.
